# MediaConvert
Media convert tool

This project is mainly used to convert media file formats, 
using the open source library referenced by JAVE2 (Java Audio Video Encoder) can be found here: 
https://github.com/a-schild/jave2 

Usage command for jar:

  <code>java -jar MediaConvert.jar "file=C:\MyVideo.avi" format=mp4 framerate=30 width=1280 height=720</code>

Usage command for jar:

  <code>"C:\MediaConvert.exe" "file=C:\MyVideo.avi" format=mp4 framerate=30 width=1280 height=720</code>

Formats can be: "mp4", "webm", "webp", "jpg", "png", "gif",...
